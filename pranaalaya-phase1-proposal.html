01

Brand Architecture

This is the most important strategic decision in this document — and it shapes everything that follows. Getting the brand structure right now means every rupee spent on marketing builds something permanent.

The client's immediate priority is clear: focus on Pranaalaya first — generate leads and awareness for the therapy centre at Karimanal and Manacaud. That is the operational goal and we will execute on it fully. At the same time, we want to share a brand architecture suggestion that adds long-term value without slowing down anything in the short term.

Our suggestion: build Alaya as the parent brand — a single icon, a shared visual language — and let Pranaalaya, Poorna Annapoorna, and any future verticals grow under it with their own colour and personality. This way, the brand you build for Pranaalaya today automatically becomes equity for everything that comes after.

Shared Brand System — What Stays Constant Across All Verticals

Icon

One Alaya parent mark — a single abstracted symbol rooted in Kerala / Ayurvedic tradition. All three brands use this icon. Only the colour changes.

Typography

One serif (Cormorant or similar heritage face) + one clean sans. Consistent across all three brands — only weight and application varies.

Colour

Each vertical has its own primary and secondary colour palette. Alaya's gold threads through all — the connective identity signal.

Imagery

Each brand has its own photography direction, tone, and content style — clinical for Pranaalaya, earthy for Poorna Annapoorna, architectural for Alaya.

On the Client's Priority — Pranaalaya First, Absolutely

This brand architecture doesn't slow down the Pranaalaya launch by a single day. We design the Alaya parent icon and the Pranaalaya sub-brand together as part of the branding package — it takes the same amount of time as designing Pranaalaya alone, and you end up with a system that works for all three brands. The only difference is we're building a foundation instead of a one-off logo.

An Honest Observation

With respect — the two-week wait for the shoot doesn't actually need to pause anything. We need roughly one week to complete the brand guidelines, set up GMB, install ad pixels, build the content calendar, and create the first round of graphic-based content (treatment posters, doctor introduction, package cards). That's all work that has to happen regardless of Amal's shoot date.

So if we start at ₹32,000 today, we use Week 1 for all the foundation work, Week 2 for first posts and ad campaigns — and when Amal delivers the video files, they slot directly into a content system that's already optimised and running. The video component of the monthly content count would still be fully met once Amal's edits are in. The only difference is we'd be three weeks ahead on lead generation.

That said, we understand if the phased approach feels more comfortable. The ₹18,000 starter plan below is a solid beginning — and the upgrade path to ₹32,000 is clearly defined.
